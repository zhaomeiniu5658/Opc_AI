# 字体规范

移动端页面使用系统无衬线字体。首页文本以高对比、紧凑的工具型层级组织，字距保持默认值，不使用负字距或营销式超大标题。

- “我的简历”入口：`22rpx`，字重 `800`
- 首屏问候语：`36rpx`，字重 `900`，行高 `1.28`
- 首屏主标题：`52rpx`，字重 `900`，行高 `1.1`，亮紫色并带轻微发光

- 在线状态：`24rpx`

## 文本规则

- 按钮、导航与功能入口使用短文本，保持单行可读。
- 品牌名、文件名和联系方式超出可用宽度时使用省略号，不推动相邻控件。
- 主标题使用白色和亮紫色区分，说明文本使用蓝灰色，不以降低字号代替信息层级。
- 窄屏时优先调整布局或换行；仅首页品牌名和首屏主标题按现有响应式规则缩小。
