# 间距规范

移动端页面外边距使用 `28rpx`，首页顶部可增加到 `44rpx` 以适配自定义导航和状态栏。

## 卡片与模块

- 玻璃卡片内边距：`24rpx` 到 `28rpx`
- 卡片之间的垂直间距：`26rpx` 到 `30rpx`
- 搜索框、按钮和输入区高度：`76rpx` 到 `82rpx`
- 四宫格入口图标：`64rpx`
- 强 CTA 卡片：内边距 `28rpx`，圆角 `18rpx`

## 圆角

- 主卡片：`18rpx`
- 输入框和按钮：`14rpx` 到 `18rpx`
- 标签：`10rpx`
- 圆形状态和分数徽章：`999px`

## 底部导航

底部导航采用固定悬浮容器，左右留白 `22rpx`，底部留白 `22rpx` 并叠加 `env(safe-area-inset-bottom)`。
