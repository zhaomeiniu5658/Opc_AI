# 颜色规范

项目采用“AI 数字人求职助手”深色紫蓝视觉体系，参考界面为近黑蓝背景、紫蓝高光、玻璃质感卡片和发光行动按钮。

## 基础色

- 页面背景：`#020817`
- 次级背景：`#061122`
- 主文字：`#f8fafc`
- 辅助文字：`#aeb8d0`
- 弱提示文字：`#94a3b8`

## 品牌与强调色

- 主紫色：`#7c5cff`
- 深紫色：`#5b2eff`
- 亮紫文字：`#8f7cff`
- 辅助蓝色：`#7dd3fc`
- 在线/成功：`#5ee7a8`
- 警告：`#f59e0b`
- 危险：`#f87171`

## 面板与边框

- 玻璃面板：`rgba(18, 30, 60, 0.78)`
- 深色输入面板：`rgba(4, 10, 24, 0.72)`
- 默认边框：`rgba(148, 163, 184, 0.18)`
- 紫色强调边框：`rgba(124, 92, 255, 0.42)`

## 渐变

- 主按钮：`linear-gradient(135deg, #7c5cff 0%, #5b2eff 100%)`
- 页面背景：`linear-gradient(180deg, #020817 0%, #061122 58%, #020817 100%)`
- 强 CTA 卡片：紫色径向光晕叠加深色玻璃面板。
