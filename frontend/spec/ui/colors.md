# 颜色规范

首页是当前 `AI 数字人求职助手` 的视觉基准。整体采用近黑蓝画布、低透明紫色网格、紫蓝品牌高光和深色玻璃面板，避免大面积纯紫或浅色卡片。

## 页面画布

- 页面渐变起点：`#020816`
- 页面渐变中段：`#06152a`
- 页面渐变终点：`#020817`
- 全局布局底色：`#07111f`
- 顶部紫色环境光：`rgba(117, 92, 255, 0.12)`
- 背景网格横线：`rgba(124, 92, 255, 0.07)`
- 背景网格竖线：`rgba(124, 92, 255, 0.05)`

首页背景使用两层渐变：

```css
background:
  linear-gradient(130deg, rgba(117, 92, 255, 0.12) 0%, rgba(4, 10, 24, 0) 32%),
  linear-gradient(180deg, #020816 0%, #06152a 48%, #020817 100%);
```

## 文字颜色

- 强标题、品牌名和关键数据：`#ffffff`
- 正文和消息文字：`#f8fafc`
- 首屏说明和联系信息：`#c8d2ef`、`#cbd5e1`
- 次级说明：`#b8c2da`、`#aeb8d0`
- 弱提示、未选中导航：`#94a3b8`

## 品牌与状态色

- 主紫色：`#7c5cff`
- 深紫色：`#5b2eff`
- 亮紫色和当前导航项：`#8f7cff`
- 品牌按钮紫：`#9b7cff`
- 角色标签紫：`#a78bfa`
- 技能标签紫：`#b5a7ff`
- 在线/成功：`#5ee7a8`
- 辅助蓝色：`#7dd3fc`
- 警告：`#f59e0b`
- 危险：`#f87171`

## 面板、边框与阴影

- 玻璃面板顶部：`rgba(18, 30, 60, 0.78)`
- 玻璃面板底部：`rgba(8, 15, 31, 0.88)`
- 深色按钮/状态面板：`rgba(12, 22, 43, 0.84)`
- 次级按钮面板：`rgba(20, 30, 58, 0.78)`
- 默认边框：`rgba(148, 163, 184, 0.18)`
- 强 CTA 边框：`rgba(124, 92, 255, 0.44)`
- 底部导航边框：`rgba(99, 102, 241, 0.24)`
- 面板内高光：`rgba(255, 255, 255, 0.05)`
- 深色投影：`rgba(0, 0, 0, 0.24)` 到 `rgba(0, 0, 0, 0.38)`

## 渐变

- 品牌标识与功能图标：`linear-gradient(145deg, #7e64ff 0%, #5830f1 100%)`
- 品牌立方体：`linear-gradient(145deg, #7d61ff 0%, #4d2bdf 100%)`
- 主行动按钮：`linear-gradient(135deg, #9b7cff 0%, #5b2eff 100%)`
- 智能优化卡片：`linear-gradient(90deg, rgba(105, 77, 255, 0.72) 0%, rgba(45, 31, 115, 0.62) 38%, rgba(10, 17, 35, 0.94) 100%)`
- 简历纸张：`linear-gradient(180deg, #fbfdff 0%, #eaf2ff 100%)`

紫色只用于品牌、可操作入口、当前状态和关键数据。正文、页面底色和大面积容器保持深蓝黑，确保信息层级清楚。
