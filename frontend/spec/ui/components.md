# 组件规范

公共视觉组件位于 `src/components/common`，数字人形象位于 `src/components/digital-human`。首页通过 `DefaultLayout` 承载公共框架，页面内容本身只组合业务内容和展示组件。

## 玻璃面板

首页的消息气泡、四宫格、简历预览和最近记录使用 `glass-panel` 视觉语义：

- 背景从 `rgba(18, 30, 60, 0.78)` 过渡到 `rgba(8, 15, 31, 0.88)`。
- 边框为 `1px solid rgba(148, 163, 184, 0.18)`。
- 使用轻微内高光和向下的深色投影。
- 主面板圆角为 `18rpx`，不叠套卡片。

## 顶部品牌栏

顶部由模拟系统状态区、品牌区和“我的简历”入口组成。

- 品牌标识使用紫色立方体，不使用图片图标。
- 品牌名使用白色粗体；右侧入口为深色半透明按钮，带文档线框图标。
- 顶部内容必须为状态栏预留安全高度，不能被设备状态栏遮挡。

## AI 首屏与数字人

首屏左侧为问候语、紫色主标题、说明和在线状态，右侧为 `DigitalHumanAvatar`。

- 数字人组件由首页和数字人页面复用，组件只负责形象展示。
- 首页展示区域为 `392rpx x 572rpx`，位于首屏右下方。
- 首屏标题使用亮紫色与轻微紫色发光；在线状态使用绿色圆点和深色半透明胶囊。
- 文案层必须位于数字人视觉层之前，避免文字被形象遮挡。
- 数字人形象不绘制装饰环形边框，保留人物本体和底部渐隐效果。

## 助手消息

消息区由紫色小立方体头像和玻璃消息气泡组成。

- 头像尺寸为 `58rpx`，圆角 `14rpx`。
- 消息气泡使用 `22rpx 26rpx` 内边距，最大宽度 `520rpx`。
- 文本使用白色高对比正文，不将帮助说明放入独立说明卡片。

## 四宫格能力入口

能力入口固定为四列：优化建议、岗位匹配、面试辅导、求职攻略。

- 外层为一个玻璃面板，内部四项以细分隔线区分，不拆为四张独立卡片。
- 每项包含紫蓝渐变图形、主标题和一行说明。
- 图形尺寸为 `66rpx`，标题与说明居中排列。

## 简历预览与记录

- 简历预览使用浅色纸张缩略图，和深色信息区形成明确对比。
- 姓名、职位、联系方式、摘要和技能标签按信息密度由强到弱排列。
- 最近记录使用单行文件条目、文件图标、时间/得分和右箭头；长文件名必须省略，不换行挤压布局。

## 智能优化 CTA

CTA 使用紫色到深蓝的横向渐变，左侧为魔杖图形，中部为标题与一句说明，右侧为紫色主操作按钮。

- CTA 圆角 `18rpx`，带紫色强调边框。
- 主操作按钮最小高度 `76rpx`，保持短文本和箭头组合。

## 底部导航

`AppTabBar.vue` 是唯一底部导航实现，由 `DefaultLayout.vue` 引入。

- 固定贴底、满宽显示，不参与页面滚动布局。
- 五项固定为：首页、数字人、岗位、成长、我的。
- 五项分别直达 `/pages/index/index`、`/pages/digital-human/index`、`/pages/job/list`、`/pages/career/roadmap`、`/pages/profile/index`，不使用首页查询参数切换内容。
- 菜单切换优先复用已访问页面，避免整页重建造成的白屏；首次访问再创建目标页面。
- 使用 CSS 图形图标和文字组合，不依赖官方 `tabBar` 图标资源。
- 未选中项使用 `#94a3b8`，当前项使用 `#8f7cff`。
- 页面滚动区需预留导航栏与安全区空间，避免最后一项内容被遮挡。
