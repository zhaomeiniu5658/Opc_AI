# 页面布局规范

页面第一屏直接展示可操作内容，不做营销落地页。首页定位为 `AI 数字人求职助手`，视觉结构参考深色移动端助手界面。

## 首页

首页从上到下依次为：

1. 品牌栏：左侧品牌标识和“AI数字人求职助手”，右侧“我的简历”入口。
2. AI 助手首屏：欢迎文案、紫色 “AI求职助手” 主标题、在线状态和右侧数字人形象。
3. 助手消息：头像图标 + 玻璃消息气泡，提示用户上传或编辑简历。
4. 搜索区：深色输入框 + 紫蓝渐变搜索按钮。
5. 四宫格能力入口：优化建议、岗位匹配、面试辅导、求职攻略。
6. 简历预览：纸张缩略图、姓名职位、联系方式、摘要和能力标签。
7. 强 CTA：智能优化简历。
8. 最近优化记录和推荐岗位。
9. 底部导航：深色玻璃底栏，当前项紫色高亮。

业务页面统一支持 `loading`、`empty`、`error`、`ready` 四种状态；状态页也应继承紫蓝深色视觉体系。
