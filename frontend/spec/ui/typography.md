# 字体规范

移动端页面使用系统无衬线字体。字体层级强调清晰、紧凑、偏工具型，不使用过大的营销式段落。

## 首页层级

- 品牌名：`34rpx`，字重 `900`
- 首屏问候语：`40rpx`，字重 `900`
- 首屏关键词标题：`56rpx`，字重 `900`，使用紫色和轻微发光
- 模块标题：`30rpx` 到 `34rpx`，字重 `900`
- 正文说明：`24rpx` 到 `28rpx`，行高 `1.5` 到 `1.65`
- 辅助说明和标签：`22rpx` 到 `24rpx`

## 文本规则

按钮文案保持短句，避免超过一行。卡片标题可以加粗但不使用负字距。标签、分数、状态徽章使用高对比颜色，确保在深色玻璃背景上可读。
